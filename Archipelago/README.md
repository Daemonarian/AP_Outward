# Outward: Definitive Edition APWorld

This project creates an *[Archipelago](https://archipelago.gg/)* APWorld for the game *Outward: Definitive Edition*.
Please see the [Archipelago documentation](https://archipelago.gg/tutorial/) for instructions on how to set up and use APWorlds.
It is intended to be used with the *[Outward Archipelago](https://github.com/Daemonarian/AP_Outward/tree/main/Outward)* mod for *Outward: Definitive Edition*.
